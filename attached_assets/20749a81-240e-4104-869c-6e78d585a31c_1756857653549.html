1. Kindroid API Setup

Configure your Kindroid companion for voice interactions.

Steps:
  1. Go to your Kindroid dashboard
  2. Navigate to Settings → API Access
  3. Generate a new API key
  4. Copy your bot ID from the companion settings
  5. Enter both values in the Credentials section
Completed

2. Eleven Labs Voice Setup

Configure text-to-speech with affordable voice options.

Steps:
  1. Create account at elevenlabs.io
  2. Choose the Starter plan (most affordable)
  3. Go to Profile → API Keys
  4. Generate and copy your API key
  5. Browse voice library and copy desired voice ID
Completed

3. Spotify Developer Setup

Enable music integration for your companion.

Steps:
  1. Go to developer.spotify.com
  2. Log in with your Spotify account
  3. Create a new app in the dashboard
  4. Copy Client ID and Client Secret
  5. Add redirect URI: http://localhost:3000/callback
In Progress

4. Twilio Voice & SMS Setup

Configure calling functionality and SMS management.

Steps:
  1. Sign up at twilio.com (free trial available)
  2. Verify your phone number
  3. Purchase a phone number for calls/SMS
  4. Get Account SID and Auth Token from console
  5. Configure webhook URLs for voice and SMS

Common Issues & Solutions

API Connection Failures
  • • Check that API keys are correctly entered
  • • Verify your account has sufficient credits/quota
  • • Test connection using the diagnostic tools
  • • Check firewall settings for outbound API calls
Voice Quality Issues
  • • Try different Eleven Labs voice models
  • • Check your internet connection stability
  • • Adjust audio settings in browser
  • • Consider upgrading Eleven Labs plan for better quality
SMS Not Working
  • • Verify phone number format (+1234567890)
  • • Check Twilio account balance and status
  • • Ensure SMS consent checkbox is checked
  • • Test with simple commands like "status"
